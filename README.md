# PI Source Reading

A dedicated source-reading repository for studying the `pi-mono` codebase and related runtime architecture.

## Scope
- interactive coding-agent host architecture
- extension runtime
- session runtime
- subagent implementation patterns
- future package-by-package source reading notes

## Initial structure
- `_meta/` — repo index and reading map
- `notes/runtime/` — runtime architecture notes
- `notes/extensions/` — extension loader/runner/API notes
- `notes/subagents/` — subagent implementation notes
- `raw/` — raw captures, excerpts, and supporting materials
- `assets/` — diagrams or exported figures

## Current focus
- `pi-mono` (`badlogic/pi-mono`)
